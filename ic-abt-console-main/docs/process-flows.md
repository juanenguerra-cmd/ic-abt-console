# Data Entry Process Flows

This guide outlines the step-by-step process for entering key infection control events in the ICN Console.

---

## 1. Infection Prevention (IP) Event

**Purpose:** Document a new infection, symptom onset, or precaution requirement for a resident.

### Process Flow:
1. **Navigate to Resident Board:** Click on "Resident Board" in the main navigation or sidebar.
2. **Select Resident:** Find the resident using the search bar or by filtering by unit. Click on the resident's card to open their profile.
3. **Initiate IP Event:** In the resident's profile, click the **"New IP Event"** button.
4. **Complete the IP Event Form:**
   - **Infection Details:** Select the suspected or confirmed infection type (e.g., COVID-19, Influenza, UTI).
   - **Symptoms:** Log any active symptoms and their onset date.
   - **Precautions:** Select the required isolation precautions (e.g., Contact, Droplet, Airborne).
   - **Notes:** Add any relevant clinical context.
5. **Save:** Click "Save IP Event" to record the event. The resident's status will update automatically on the Resident Board and Dashboard.

## 2. Antibiotic (ABT) Event

**Purpose:** Track antibiotic prescriptions, indications, and review dates for stewardship.

### Process Flow:
1. **Navigate to Resident Board:** Go to the "Resident Board".
2. **Select Resident:** Open the profile of the resident receiving the antibiotic.
3. **Initiate ABT Event:** Click the **"New ABT Course"** button.
4. **Complete the ABT Form:**
   - **Medication:** Enter the antibiotic name, dose, and route.
   - **Indication:** Select the reason for the prescription (e.g., UTI, Pneumonia, Skin Infection).
   - **Dates:** Enter the start date and the planned end date.
   - **Review Date:** Set a date for the 72-hour antibiotic timeout/review.
5. **Save:** Click "Save ABT Course". The course will now appear in the resident's profile and the Antibiotic Stewardship reports.

## 3. Vaccination Event

**Purpose:** Record a resident's vaccination status (e.g., COVID-19, Influenza, RSV, Pneumococcal).

### Process Flow:
1. **Navigate to Resident Board:** Go to the "Resident Board".
2. **Select Resident:** Open the resident's profile.
3. **Initiate Vaccination Event:** Click the **"Log Vaccine"** button.
4. **Complete the Vaccination Form:**
   - **Vaccine Type:** Select the vaccine administered or offered.
   - **Status:** Choose the outcome (e.g., Administered, Refused, Medical Contraindication).
   - **Date:** Enter the date of administration or refusal.
   - **Details:** Add lot number, manufacturer, and administration site if applicable.
5. **Save:** Click "Save Vaccine Record". This updates the facility's vaccination coverage metrics.

## 4. Contact Tracing Event

**Purpose:** Identify and monitor residents or staff exposed to an infectious disease.

### Process Flow:
1. **Navigate to Outbreaks:** Click on "Outbreaks" in the sidebar.
2. **Select or Create Outbreak:** Open an active outbreak or create a new one.
3. **Initiate Contact Tracing:** Within the outbreak dashboard, navigate to the "Contact Tracing" tab and click **"Add Exposure"**.
4. **Log Exposure Details:**
   - **Exposed Individual:** Select the resident or staff member who was exposed.
   - **Source:** Link the exposure to a known index case if applicable.
   - **Exposure Date:** Enter the date of last known exposure.
   - **Risk Level:** Assess the risk (e.g., High, Low).
   - **Monitoring:** Set the required monitoring period (e.g., 14 days) and any required testing dates.
5. **Save:** Click "Save Exposure". The individual will be added to the contact tracing line list for daily monitoring.

## 5. Line Listing Event

**Purpose:** Generate and manage a line list of cases and exposures during an outbreak.

### Process Flow:
1. **Navigate to Line List Report:** Click on "Line List Report" in the sidebar under Surveillance.
2. **Select Outbreak/Criteria:** Choose the specific outbreak or filter criteria (e.g., date range, unit, infection type) to generate the list.
3. **Review Line List:** The system automatically compiles data from IP Events and Contact Tracing into a tabular format.
4. **Update Statuses:** Directly from the line list, you can update resident statuses, add daily symptom checks, or log new test results.
5. **Export/Print:** Use the "Export to CSV" or "Print" buttons to share the line list with public health authorities or facility leadership.
